\# SkillTreeBattleDemo - 2D战斗原型



> 一款基于 Unity 2D 的技能树战斗 Demo，实现了角色操控、敌人 AI、技能树框架与自定义法术系统。



\## 🎮 操作说明



| 按键 | 功能 |

|------|------|

| W/A/S/D | 角色移动 |

| J | 普通攻击（需在技能树中解锁劈砍技能） |

| U | 燃烧法术（在角色位置放置火焰区域,需在技能树中解锁法术） |

| E | 打开/关闭技能树窗口 |



\## ✨ 已实现功能



\- \*\*地图系统\*\*：Tilemap 瓦片地图，含可通行山地碰撞，场景切换触发器

\- \*\*角色系统\*\*：八方向移动，动画状态机

\- \*\*战斗系统\*\*：J 键普攻（技能树解锁后方可使用），伤害与血量结算

\- \*\*敌人 AI\*\*：追踪范围检测、移动靠近、近战攻击、受击反馈

\- \*\*技能树系统\*\*：可扩展框架，含两种技能节点

&#x20; - 🔪 劈砍解锁（前置技能，解锁后可使用 J 键攻击）

&#x20; - ❤️ 生命上限提升

\- \*\*法术系统\*\*：燃烧技能，U 键在当前位置生成火焰区域，敌人进入扣血

\- \*\*UI 系统\*\*：血量条、经验值、属性面板、技能树窗口、技能框占位

\- \*\*数值闭环\*\*：经验获取、血量变化、技能点消耗全部驱动 UI 刷新
